## 프로젝트 소개 

- 전시회와 팝업 스토어 정보 한번에 찾아볼수 있는 웹 사이트 입니다.
- sns만을 보게 되더라도 많은 사람들이 전시회,팝업 스토어에서 시간을 보냅니다.
  
-**Diggingplace 라는 웹 사이트를 만들게 된 계기이기도 하며
-**한번에 두 컨턴츠의 정보를 얻을수 있다면 좋겠다 라는 생각에 프로젝트를 진행하였습니다.
