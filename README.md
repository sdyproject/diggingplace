## 프로젝트 소개 

- 전시회와 팝업 스토어 정보 한번에 찾아볼수 있는 웹 사이트 입니다.

  
> **특정 장소에서 특별한 경험을 할 수 있는 팝업 스토어, 전시회가 많이 진행되고 있습니다.**
> **한 번에 두 컨텐츠의 정보를 얻을수 있다면 좋겠다는 생각에 프로젝트를 진행하였습니다.**
